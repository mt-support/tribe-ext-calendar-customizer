Customize the looks of The Events Calendar's month view.

== Description ==

This extension adds more than a dozen formatting options to the WordPress customizer that will enable you to customize the looks of the month view in The Events Calendar. You will find the settings under Themes > Customize > The Events Calendar > Month View.
